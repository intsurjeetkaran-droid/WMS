🚀 HOW TO RUN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BACKEND:
  1. Make sure MongoDB is running locally (port 27017)
  2. cd backend
  3. npm run dev
  → API runs at http://localhost:5000

FRONTEND:
  1. cd frontend
  2. npm run dev
  → App runs at http://localhost:5173

FIRST LOGIN:
  1. Go to http://localhost:5173
  2. Click "Get Started" → Register
  3. First user gets 'viewer' role by default
  4. To make super_admin: update role in MongoDB directly or via API
